📘 User Education

🌍 Why we created this app?

We live in a digital world where your data is your identity. Keeping it safe is critical.

🔐 What is Cybersecurity?

Cybersecurity means protecting your files, privacy, and identity from unauthorized access 👨‍💻.

🛡 Why is it important?

Hackers 🕵️ can misuse your files. This app ensures only you and those you trust can view your files.

⚡ How this app safeguards you?

  • 🔑 Hybrid Encryption → AES + RSA
  • 🧩 Each file is locked with a unique AES-GCM key + initialization vector (IV).
  • 📤 AES key is encrypted with RSA Public Key of the recipient.
  • 📥 Recipient unlocks it using their RSA Private Key.

🚨 Be Aware of Cyber Attacks

  • 🎣 Phishing Emails → Fake mails tricking you to share passwords.
  • 💻 Malware → Harmful files disguised as normal downloads.
  • 👀 Data Breaches → Hackers stealing data from weak apps.
  • 💳 Financial Fraud → Hackers may steal money if they get your info.

💡 How to Stay Safe?

  • 🔒 Always use strong, unique passwords.
  • 📱 Enable Two-Factor Authentication (2FA).
  • 🚫 Don’t click unknown links or attachments.
  • 🌐 Share files only with people you trust.
  • 📶 Avoid logging in from public Wi-Fi.

📂 Example of how this application secure your files

If a single user is involved

- Alice encrypts her file with AES-GCM 🔒.
- AES key itself is locked using Alice RSA Public Key 📡.
- Only Alice can unlock using her RSA Private Key 🔑.
- ✅ Ensures trust & security.

If multiple users are involved

- Alice encrypts her file with AES-GCM 🔒.
- AES key is locked using RSA Public Key of Alice 📡.
- If Alice wants to share her file with another user, her AES key is decrypted with her RSA private key and re-encrypted with the receiver's public key 🔑.
- ✅ Ensures secure sharing.

🎯 Summary

Your files are always encrypted. Sharing = safe. Trust = guaranteed. 🔐✨
Stay smart, stay secure, and protect your digital identity 🌍.
